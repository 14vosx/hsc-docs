# Audit — README

## Objetivo

Esta pasta existe para concentrar documentos de auditoria, pendências abertas, inconsistências detectadas e pontos que ainda exigem reconciliação explícita no sistema documental HSC.

Ela serve para:

- registrar open items de migração documental
- concentrar gaps identificados durante revisão
- manter rastreabilidade de pendências estruturais
- apoiar revisão periódica de consistência entre documentos
- evitar que observações de auditoria se percam no fluxo normal de implementação

## Regra de leitura

Esta pasta não é fonte canônica principal de operação.

A leitura correta é:

1. entrar primeiro no contexto canônico correspondente
2. usar auditoria para localizar pendências, desvios ou reconciliações necessárias
3. retornar ao documento canônico depois de entender o open item

---

## Organização

Os arquivos desta pasta devem permanecer preferencialmente:

- datados no nome
- explícitos sobre o problema observado
- focados em pendência, inconsistência ou reconciliação
- escritos para leitura rápida e revisão periódica no Obsidian

Este `README.md` funciona como porta de entrada e índice local do contexto.
